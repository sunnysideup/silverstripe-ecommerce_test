ViewableData::ThemeDir

Return the directory if the current active theme (relative to the site root).

This method is useful for things such as accessing theme images from your template without hardcoding the theme page - e.g. <img src="$ThemeDir/images/something.gif">. This method should only be used when a theme is currently active. However, it will fall over to the current project directory.
